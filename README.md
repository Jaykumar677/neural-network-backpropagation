# Neural Network Backpropagation

## Overview

This project demonstrates the implementation of a neural network training process from scratch using NumPy. The project focuses on understanding the mathematical foundations of backpropagation, reverse-mode differentiation, Jacobian composition, and gradient verification without relying on automatic differentiation frameworks.

## Features

- Forward propagation
- ReLU activation function
- Backpropagation from scratch
- Reverse-mode differentiation
- Jacobian pullbacks
- Gradient computation for network parameters
- Finite-difference gradient checking
- Fully vectorised NumPy implementation

## Technologies Used

- Python
- NumPy
- Jupyter Notebook

## Learning Objectives

This project explores:

- Neural network fundamentals
- Chain rule and Jacobian composition
- Vector-Jacobian Products (VJP)
- Reverse-mode automatic differentiation
- Gradient verification techniques
- Numerical optimisation concepts
